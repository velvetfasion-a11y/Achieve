# Achieve

Achieve is a React Native + Expo app for habit tracking, AI-guided motivation, and private high-frequency journaling.

## App Summary

- **App Name:** Achieve
- **Subtitle:** Your AI Coach for High Frequency
- **Primary Category:** Health & Fitness
- **Secondary Category:** Productivity

## Included Features

- 3-tab app flow: **Achieve**, **AI Coach**, **Profile**
- Empty-first habit tracker with suggestions and category tagging
- Live radial "Today's Frequency" progress ring
- AI Coach with 3 modes: Motivation, Routines, Energy
- Persistent local memory for conversations, habits, notes, photos, settings
- Notes to Self with generated affirmation button
- Journal Photos stored full-size with full-screen zoom
- Custom accent colors (presets + hex input)
- Profile dashboard with lifetime frequency and day streak
- Settings for reminder frequency, notification style, export format, default coach mode
- One-tap JSON/CSV export
- Anonymous local leaderboard view
- Sign-out / local wipe control

## Tech Stack

- Expo
- React Native
- React Navigation (Bottom Tabs)
- AsyncStorage (offline-first local persistence)
- Expo Image Picker, File System, Sharing, Blur
- react-native-svg
- Inter font (@expo-google-fonts/inter)

## Run Locally

```bash
npm install
npm run start
```

Then open with Expo Go or iOS/Android simulator.
